Política de Privacidad

Última actualización: 2 de julio de 2026

Esta Política de Privacidad explica cómo el bot de Discord recopila, utiliza y protege la información necesaria para proporcionar sus funciones. Al utilizar el bot, aceptas las prácticas descritas en este documento.

1. Información recopilada

El bot puede almacenar únicamente la información necesaria para su funcionamiento, incluyendo:

  • ID de usuario de Discord.
  • ID del servidor (Guild).
  • ID de canales y roles cuando sea necesario.
  • Configuraciones del servidor.
  • Comandos utilizados y registros técnicos para depuración.

El bot no recopila contraseñas, correos electrónicos, mensajes privados ni otra información personal fuera de la proporcionada por la API de Discord.

2. Uso de la información

Los datos recopilados se utilizan únicamente para:

  • Proporcionar las funciones del bot.
  • Guardar configuraciones personalizadas.
  • Corregir errores.
  • Mejorar el rendimiento.
  • Prevenir abusos del servicio.

3. Compartición de datos

No vendemos, alquilamos ni compartimos información con terceros, salvo cuando sea necesario para operar el servicio (por ejemplo, servicios de alojamiento o bases de datos) o cuando la ley lo exija.

4. Conservación de datos

Los datos permanecerán almacenados únicamente durante el tiempo necesario para el funcionamiento del bot o hasta que sean eliminados a solicitud del usuario o administrador, cuando sea técnicamente posible.

5. Seguridad

Se aplican medidas razonables para proteger la información frente a accesos no autorizados, alteraciones o pérdidas.

6. Derechos de los usuarios

Los usuarios pueden solicitar:

  • Acceso a sus datos.
  • Corrección de información.
  • Eliminación de los datos almacenados.

7. Cambios en esta política

Esta Política de Privacidad podrá modificarse en cualquier momento para adaptarse a cambios legales o mejoras del servicio. La fecha de actualización indicará la versión vigente.

8. Contacto

Si tienes alguna pregunta o deseas ejercer alguno de tus derechos, puedes contactar con el desarrollador del bot mediante el servidor oficial de soporte o el medio de contacto indicado por el desarrollador.
