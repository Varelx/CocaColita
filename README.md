# CocaColita

CocaColita es un bot de Discord diseñado para ofrecer herramientas útiles, automatizaciones y funciones para mejorar la experiencia de los servidores.

## Características

- ⚡ Rápido y ligero.
- 🛠️ Configuración sencilla.
- 🔒 Gestión segura de datos.
- 📈 Preparado para futuras ampliaciones.
- 🤖 Compatible con la API de Discord.

## Estructura del proyecto

```
CocaColita/
├── privacy.html      # Política de Privacidad
├── terms.html        # Condiciones del Servicio
└── README.md
```

## Documentación legal

Antes de utilizar el bot, consulta la documentación correspondiente:

- **Política de Privacidad:** `privacy.html`
- **Condiciones del Servicio:** `terms.html`

## Datos recopilados

El bot únicamente almacena la información necesaria para su funcionamiento, como:

- ID de usuario.
- ID del servidor.
- ID de canales y roles cuando sea necesario.
- Configuración del servidor.
- Registros técnicos para diagnóstico y mejora del servicio.

No se recopilan mensajes privados, contraseñas ni información personal fuera de la proporcionada por la API de Discord.

## Uso

Al añadir CocaColita a un servidor o utilizar cualquiera de sus funciones, aceptas las Condiciones del Servicio y la Política de Privacidad incluidas en este repositorio.
